    簡單易懂的 Python Plugin 架構 (大概 ...)    Posted: 2013-08-25 15:26

    學習用 Python 寫測試工具快兩個星期了，主要使用 clime + ucltip 跟 pyinstaller。 可是目前遇到一個問題，當我使用 pyinstaller 將測試工具打包之後， 他會很貼心的幫你將所有 .py 檔包裝成一個可執行檔。 所以當我要任何新增 command 都必須要重新部屬那個執行檔，這樣實在非常的麻煩。

    因此才會開始嘗試是否有可能執行檔只包裝 clime, ucltip 以及我自己所提供的 API， 然後 command 的部份就利用 plugin 的方式來增加。
